# Multi-Head Attention
Single head attention calculates scores (Q @ K.T) then attention_scores using softmax and finally  turn that into its output by multiplying with V.                                                               
A sentence is a complex thing, it is not only grammar but contains many things including grammar, meaning, rhythm and other things. So a single head attention cannot capture all the details so we use multiple attention heads to capture everything a sentence has to offer. We do this by reducing the dimension by dividing the dimension equally across multiple heads. So if our original dimension is 8 and we have 4 heads then dimension for each head's weight matrix is (8,2). Each attention has its own weight matrices then each calculates its Q, K, V and gives final output. After calculating attention from each head we concatenate everything. Now the vector is rich covering different aspects but they are not linked to each other. So we use another Weight_matrix (WO) that is multiplied with concatenated output to give the final output that is able to find different aspects of the sentence.
