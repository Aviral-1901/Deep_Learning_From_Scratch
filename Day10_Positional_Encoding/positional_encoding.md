# Positional Encoding
Attention is order-blind. So we need positional encoding to fix the order problem. Position encoding adds position information directly to word vectors before attention runs. We use sine/cosine waves for positional encoding. Sine and cosine both because sin0 is always 0 and using cosine ensures position0 is never a zero vector. Sine and cosine are used because they are always between -1 and 1. The values are unique per position and same value for same position regardless of length. We use sine for even dimension and cosine for odd dimension and different frequencies per dimension. So the word(X) with positional encoding is X_with_position = X + PE (word and positional encoding in one vector)
